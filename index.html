🐦 Junco Survivor Game

Survive environmental challenges, reproduce, pass on inherited boldness alleles, and track CORT stress hormone levels.

Junco Survivor: Student Instructions

Your mission: Use evidence from the simulation to explain how different environments can change junco populations over generations.

1. Form your research team

Work in a group of 2–4 students. Your group will study both populations: Urban UCSD juncos and Mt. Laguna juncos.

2. Understand the goal

Your goal is to collect evidence and explain how selection pressures affect survival, reproduction, inherited alleles, boldness, and CORT.

  • Urban goal: Look for evidence that boldness may help birds survive and reproduce near people.
  • Mt. Laguna goal: Look for evidence that cautious behavior may help birds survive and reproduce near predators and cover.

3. Set up the simulation

  1. Choose the number of generations.
  2. Choose the starting number of bold birds.
  3. Choose the starting number of shy birds.
  4. Choose selection pressure intensity: mild, normal, or strong.
  5. Click Start Game. Generation 0 is the starting population.

4. Play each generation

  1. Read the environmental condition for each habitat.
  2. Choose one strategy for Urban UCSD and one strategy for Mt. Laguna.
  3. Click Run Next Generation.
  4. Read the habitat condition popup and outcome cards.
  5. Use the data table to record what happened.

5. Use the evidence tools

  • Use the Before vs. After Comparison to compare Generation 0 to the current generation.
  • Use the data table to track bold birds, shy birds, allele frequency, CORT, and strategy used.
  • Use the CORT vs. Boldness graph to describe correlation, but remember correlation does not prove causation.

6. Run the Common Garden Experiment

  1. Run at least 2 generations first.
  2. Click Run Common Garden Experiment.
  3. Read how both populations behave when placed in the same controlled environment.
  4. Use this as evidence to decide whether boldness seems inherited, learned, or influenced by both genes and environment.

7. Make your graph on paper

Put generation number on the x-axis. Choose one y-axis variable: percent bold birds, boldness allele frequency, average CORT, or population size. Make two lines: Urban UCSD and Mt. Laguna.

8. Write your scientific explanation

  • What selection pressures affected each population?
  • Which traits helped birds survive and reproduce?
  • How did allele frequencies change?
  • How did CORT relate to boldness?
  • What evidence from the Common Garden Experiment supports inheritance?

Key ideas

  • Natural selection happens when some traits help organisms survive and reproduce more than others.
  • Alleles are versions of genes that can be passed to offspring.
  • CORT is a stress hormone. In this model, bolder populations usually have lower average CORT.
  • Correlation does not prove causation.

🧪 Common Garden Experiment

Scientists tested whether boldness was inherited or learned by raising UCSD and Mt. Laguna juncos in the same controlled environment.

Controlled Variables

  • same aviary conditions
  • same food access
  • same human exposure
  • same nesting environment

Question

If both groups are raised in the same environment, will UCSD birds still behave differently from Mt. Laguna birds?
